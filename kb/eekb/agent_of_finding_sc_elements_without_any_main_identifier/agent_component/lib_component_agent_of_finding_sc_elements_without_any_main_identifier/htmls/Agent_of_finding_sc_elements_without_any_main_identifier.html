Задачей агента поиска sc-элементов без главного идентификатора является поиск в базе знаний элементов, у которых отсуствуют необходимые конструкции, в данном случае sc-элементов, имеющих системный идентификатор, но не имеющих ни одного основного. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска sc-элементов без главного идентификатора.
